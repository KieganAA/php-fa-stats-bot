Настройки bot-stats

URL твоего bot-stats сервера. Бот пришлёт его вместе с токеном после /extension_token.

Получи в боте: /extension_token в чате @AIO_FA_Stats_Bot.

Поведение

Если выключить — кнопки «＋» у лендов и пункт правого клика продолжают работать, просто пропадёт перетаскиваемая панель с подборкой.

Как пользоваться

  1. В Telegram запусти /extension_token — получишь токен и URL.
  2. Вставь сюда, нажми «Сохранить».
  3. Открой app.aio.tech — рядом с лендами появятся чекбоксы. Отметь нужные.
  4. Кликни иконку расширения → «Создать подписку» → выбери частоту → готово.

Альтернативно: выдели в любом тексте список id (33169, 205215) → правый клик → «Отправить в bot-stats». Откроется popup с уже подгруженными лендами.
